남춘애 (옆집아줌마)

  • 나이 : 56살
  • 직업 : 전업주부
  • 좋아하는 것 : 수다떠는 것, 여러 취미생활
  • 싫어하는 것 : 마을에 찾아오는 수상한 사람달
  • 특징 : 떡잎 마을의 부녀회장 / 종종 짱구네에 음식을 가져다 주시곤 하며 가끔 짱구와 짱아를 돌봐주시곤 한다.

민희 & 정훈 (닭살부부)

  • 나이 : 26 / 22살
  • 좋아하는 것 : 공통 - 서로가 서로를 가장 좋아함, 짱구네에서 밥 얻어먹기 / 민희 - 공포영화 / 정훈 - 피규어
  • 싫어하는 것 : 부부간의 의견충돌
  • 특징 : 짱구네에게 도움을 많이 청하며, 무단침입도 자주 범함 (부부싸움 및 여러 집안문제, 고민들 등등) /

유이슬

  • 나이 : 20살
  • 직업 : 대학생
  • 좋아하는 것 : 짱구를 포함한 아이들, 스모
  • 싫어하는 것 : 아버지의 과잉보호
  • 특징 : 박광자와 절친한 사이

최마리 & 김용숙 & 이은혜

  • 나이 : 18살
  • 직업 : 고등학생
  • 소속 : 붉은 장미 삼총사
  • 좋아하는 것 : 다른 조직과의 대결
  • 싫어하는 것 : 짱구가 개그스승님이라고 부를 때, 짱구가 귀찮게 굴 때
  • 특징 : 리더 김용숙을 필두로 한 3인조이며 김용숙이 구호를 외칠 때마다 나머지 팀원들은 부끄러워하기도 한다. / 김용숙은 보기와는 다르게 코스프레를 좋아하며 나머지 멤버들도 평범한 여고생의 일상을 선호하는 편이다.

오수

  • 나이 : 22살
  • 직업 : 5수 후 새내기 대학생
  • 좋아하는 것 : 미녀, 먹는 것, 짱구네에서 밥 얻어 먹을 때
  • 싫어하는 것 : 대학 입학에 관한 모든 것에 예민하며 스트레스를 받음
  • 특징 : 와르르맨션에 거주 중이며 짱구네의 예전 이웃이기도 하다. / 배가 고플 때 종종 짱구집에 가서 얻어 먹음
