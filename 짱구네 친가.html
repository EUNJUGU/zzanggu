<환> 짱구네 <영>

신짱구

  • 나이 : 5살
  • 소속 : 떡잎유치원 해바라기반
  • 직업 : 유치원생
  • 좋아하는 것 : 초코비, 액션가면, 건담로봇, 부리부리용사, 이슬이 누나
  • 싫어하는 것 : 피망
  • 별명 : 감자머리
  • 특징 : 아침마다 화장실로 인해 유치원에 거의 매일 지각함

신형만

  • 나이 : 35살
  • 소속 : 떡잎상사
  • 직업 : 회사원(계장)
  • 좋아하는 것 : 골프, 주말에 낮잠자기, 맥주, 미녀, 비상금
  • 싫어하는 것 : 주말에 아이들 돌보기, 쥐꼬리 월급
  • 별명 : 만년계장, 발냄새괴물
  • 특징 : 발냄새가 매우 지독함

봉미선

  • 나이 : 29살
  • 직업 : 전업주부
  • 좋아하는 것 : 낮잠, 보석, 쇼핑, 수다떨기, 미남, 드라마
  • 싫어하는 것 : 주말에 아이들 돌보기, 쥐꼬리 월급
  • 별명 : 삼겹살배 / 구두쇠 / 납작가슴 아줌마, 짠순이, 잔소리대마왕, 낮잠 마녀, 무쇠주먹 등등 다수의 별명소유자
  • 특징 : 할인 및 공짜를 보면 전투력이 올라감

신짱아

  • 나이 : 1살
  • 좋아하는 것 : 반짝이는 물건, 미남, 엄마 화장품, 분유
  • 싫어하는 것 : 손톱깎기, 머리감기, 미남의 관심을 뺏겼을 때
  • 별명 : 짱아공룡
  • 특징 : 불리한 상황이 왔을 때 요리조리 잘 피해다님

흰둥이

  • 성별 : 수컷
  • 주특기 : 솜사탕 변신, 배꼽 긁기, 물구나무, 훌라댄스, 엉덩이 춤
  • 좋아하는 것 : 산책, 고급사료, 예쁜 암컷 강아지, 짱구네 가족
  • 싫어하는 것 : 동물병원, 주사, 약, 벌레
  • 특징 : 혼자 알아서 집을 잘 찾아감(때론 심부름도 함), 짱구 가족의 생각보다 매우 영리함
