<환> 철수네 <영>

김철수

  • 나이 : 5살
  • 소속 : 떡잎유치원 해바라기반
  • 직업 : 유치원생
  • 좋아하는 것 : 마법소녀 프린세스, 신비한 마법소녀 마리, 액션가면
  • 싫어하는 것 : 짱구의 말썽 및 귓바람, 당근, 브로콜리, 리얼 소꿉놀이
  • 별명 : 감자머리
  • 특징 : 아침마다 화장실로 인해 유치원에 거의 매일 지각함

철수 엄마

  • 나이 : 30대
  • 직업 : 전업주부
  • 좋아하는 것 : 명품
  • 싫어하는 것 : 철수의 버릇없는 행동
  • 특징 : 아들자랑이 팔불출임

철수 아빠

  • 직업 : 샐러리맨
  • 좋아하는 것 : 낮잠, 보석, 쇼핑, 수다떨기, 미남, 드라마
  • 특징 : 직장이 호주에 있어 철수와 같이 살지 않음
